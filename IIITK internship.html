Diabetes Prediction System

An interpretable web application using stacked ensemble machine learning

Project Overview

This project, completed under the guidance of Dr. Sushitha Susan Joseph at IIIT Kottayam, focuses on early diabetes prediction using a stacked machine learning ensemble. The system integrates Random Forest, SVC, and KNN as base models with Logistic Regression as a meta-classifier, achieving 82% accuracy and an ROC-AUC score of 0.8719. It also uses LIME for explainability and is deployed via a Flask web app.

Key Features

  • Stacked model: Random Forest, SVC, KNN + Logistic Regression
  • SMOTE for class imbalance handling
  • Chi-square feature selection with SelectKBest
  • Model explainability using LIME
  • Interactive Flask web application for predictions

Technologies Used

  • Python (scikit-learn, pandas, Flask, imbalanced-learn)
  • Frontend: HTML, CSS
  • Deployment: Flask web server
