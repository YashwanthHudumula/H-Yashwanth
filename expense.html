Overview
Project Overview

This project was developed during my internship at Infosys Springboard as part of a full-stack team. The Expense Management System provides an efficient solution for tracking, managing, and analyzing expenses, ensuring transparency and automation in financial management across organizations.

I led the team and was responsible for both the backend architecture and frontend integration, managing a system that handles 100,000+ records with optimized performance and secure access control.
