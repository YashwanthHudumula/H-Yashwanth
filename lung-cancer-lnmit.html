01 Overview

What This Project Does

Lung cancer is one of the most fatal malignancies worldwide, primarily due to delayed diagnosis. This project proposes a comprehensive three-stage deep learning pipeline for automated lung cancer analysis from CT scan images.

Each stage handles a distinct task: EfficientNetB3 detects whether a tumor is present, U-Net precisely segments the tumor region, and DenseNet201 classifies it into one of four cancer types.

Grad-CAM visualizations are integrated into both detection and classification stages to provide transparent, interpretable AI predictions for clinical trust.

The 4 Cancer Classes
AdenocarcinomaMost common lung cancer subtype
Squamous CellLinked to smoking, central location
Large CellAggressive, poor prognosis
NormalNo malignant tissue detected
Dataset

LIDC-IDRI dataset — CT images labeled for tumor/normal classification, with annotated segmentation masks. Split 75:10:15 (train/val/test) for detection; ~70:15:15 for segmentation.

02 Three-Stage Pipeline

Detection → Segmentation → Classification

The pipeline is sequential and modular — each stage feeds enriched information into the next, improving overall accuracy and robustness.

Stage 01 · Detection
EfficientNetB3
Binary Classification (Tumor / Normal)
90.24%
Test Accuracy · F1: 0.9487
Compound-scaled CNN detects whether a CT scan contains a suspicious tumor region. Trained over 40 epochs (20 frozen + 20 fine-tuning) with learning rate decay. Minimal false negatives — critical in oncology.
Stage 02 · Segmentation
U-Net
Pixel-level Tumor Region Localization
99.10%
Test Accuracy
U-Net encoder-decoder with skip connections generates pixel-precise tumor masks from 256×256 grayscale CT slices. Trained 20 epochs on LIDC-IDRI. Delineates tumor boundaries for downstream classification.
Stage 03 · Classification
DenseNet201
4-Class Cancer Type Identification
94.92%
Test Accuracy · Avg F1: 0.94
DenseNet201 with ImageNet pretraining classifies segmented regions into Adenocarcinoma, Large Cell, Squamous Cell, or Normal. 22 epochs with early stopping. Strong diagonal in confusion matrix.

Grad-CAM Explainability

Applied to both EfficientNetB3 and DenseNet201. Generates heatmaps highlighting tumor-relevant regions. For tumor-positive images, activation closely matches pathological tissue. For normal cases, no significant activation — validating correct model focus.
