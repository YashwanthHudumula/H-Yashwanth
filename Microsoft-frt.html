Overview
Project Overview

Developed as part of the Microsoft Future Ready Talent (FRT) program, this e-learning platform provides students with a flexible and interactive learning environment combining both recorded and live classes.

The platform is deployed on Microsoft Azure Static Web Apps, achieving 99.9% uptime with real-time analytics, cloud-based authentication, and a seamless responsive UI for students and educators alike.
