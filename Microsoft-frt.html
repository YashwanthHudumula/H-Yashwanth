E-learning Platform

An online education system with live and recorded classes

Project Overview

This project was developed as part of the Future Ready Talent (FRT) program by Microsoft. It aims to create an e-learning platform with both recorded and live classes, providing students with a flexible and interactive learning experience.

Key Features

  • Recorded video lectures for self-paced learning
  • Live classes for real-time interaction
  • User-friendly interface with seamless navigation
  • Cloud-based deployment using Microsoft Azure

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Cloud Hosting: Azure Static Web Apps
  • Live Streaming: Integration with online meeting platforms
