Overview
Project Overview

Reliably telling drones apart from birds and irrelevant background objects is critical for airspace monitoring and counter-UAV systems, but it's a hard problem: objects are small, thumbnails are just 32×32 pixels, and the dataset is both imbalanced and tiny (~572 labelled samples).

This project introduces img_traject_combined, a model that processes three parallel streams — an EfficientNet-B0 image sequence encoder, a Transformer-based trajectory sequence encoder, and a hand-crafted kinematic feature MLP — and combines them through a learned gated multimodal fusion mechanism that adaptively weights each modality per instance.

On the Visgrid classification dataset, the model reaches a macro F1-score of 0.9392 on validation and 0.8919 on the held-out test set, with a macro AUC of 0.9789, outperforming image-only, trajectory-only and naive concatenation-fusion baselines.

Architecture
Model Pipeline
Stream 01
Image Sequence Encoder
EfficientNet-B0 (ImageNet pretrained, partial fine-tuning) over up to 16 upsampled 96×96 frames, with temporal attention pooling to a 128-d embedding.
Stream 02
Trajectory Sequence Encoder
Single-layer Transformer encoder over 2D + 3D position, velocity, acceleration and speed features, padded to length 200, with attention pooling.
Stream 03
Kinematic Feature MLP
Hand-crafted statistics — speed percentiles, curvature, vertical oscillation, FFT-based flapping frequency, path straightness — through a 3-layer MLP.
Fusion
Gated Multimodal Fusion
A softmax gating network learns per-instance weights over the three embeddings, letting the model lean on whichever modality is most informative for each object.
Head
Classification & Loss
A 2-layer MLP head trained with Focal Loss, label smoothing and class weights [1.0, 2.5, 2.0] to prioritise drone detection under class imbalance.
Post-hoc
Threshold Calibration
A grid search over per-class probability bias terms after training, correcting the tendency of neural classifiers to underestimate minority-class confidence.

Removing 3D trajectory data causes the largest test-set drop of all ablations (0.8919 → 0.8707), showing that height, angle and distance encoded in 3D trajectories carry discriminative information that 2D angular data alone cannot recover — even though the smaller validation set doesn't show the same gap.

Key Contributions
Novel Contributions

• Gated multimodal fusion over naive concatenation — a learned softmax gate adaptively weights image, trajectory and kinematic embeddings per object instance.

• FFT-based frequency features on the speed signal, capturing the periodic flapping frequency that distinguishes birds from the smoother motion of drones.

• 3D trajectory fusion from multi-camera triangulation, adding height/angle/distance information beyond single-camera 2D tracks.

• Post-hoc decision threshold calibration via grid search, correcting minority-class under-confidence without any additional training cost.
